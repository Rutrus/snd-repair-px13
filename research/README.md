## `acpi_debug/`

ACPI table dump (DSDT, SSDT, etc.) from the ProArt PX13. Useful to correlate SoundWire nodes with DisCo; several MB.

Regenerable with `acpidump`, `iasl`. `.dat` / `.dsl` files are in `.gitignore`.
